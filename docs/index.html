Usage

The basic unit for the Lenka library is a layer.

The layer is a collection of pieces of text and tags:

Each part of tag opens with the characters "{{" and closes with the characters "}}". A part of the tag has a body (can be empty), and may also contains a comment (if a comment presents, it starts with the "#" and continues to the end of the tag part):
